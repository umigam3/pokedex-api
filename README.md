# Umigame's Pokémon Database

> Basic Pokédex made with HTML, JavaScript and CSS.

## Introduction

Welcome to my [Pokémon Database](https://umigam3.github.io/pokedex-api/). I made this Pokédex mainly to learn more about API calls and how to implement those in JavaScript using <strong>Fetch API</strong>. All the data is extracted from an API called [PokéAPI](https://pokeapi.co/).

In addition to that, I'm making this little project to get knowledge about structuring HTML pages and improving them with CSS. I implemented Bootstrap to familiarize with this framework.

<img src="img/pidgeot.gif" width="200">

## Features

This is a basic PokéDex that shows basic information about all the Pokémons up to generation 7. It displays diferents parameters of each Pokémon such as the Name, Type, Total Stats, HP, Attack, Defense, Special Attack, Special Defense and Speed.

<img src="img/readme/general_view.png">

## Conclusion

This project helped me to understand more about the <strong>Bootstrap Framework</strong>, in addition to that I learned how to use basic functionalities of flex and how to create <strong>Dynamic tables</strong> with JavaScript creating DOM objects.
